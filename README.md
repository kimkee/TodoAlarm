# ⏰ 할일 알리미 (TodoAlarm)

> 지정된 요일과 시간에 맞춰 브라우저 알림을 띄우고 지정된 사이트로 바로 이동할 수 있는 프리미엄 스케줄 관리용 크롬 익스텐션입니다.

---

## 🎨 주요 특징 (Key Features)

* **주 단위 스케줄 설정**: 월요일부터 일요일까지 원하는 요일(다중 선택 가능)과 시각을 지정하여 매주 반복 알림을 설정할 수 있습니다.
* **이동할 사이트 URL 연동**: 알림을 클릭하거나 알림 내의 **[사이트 열기]** 버튼을 누르면 설정한 URL이 새 탭으로 즉시 열립니다.
* **10분 뒤 다시 알림 (Snooze)**: 지금 바쁘다면 알림창의 **[10분 뒤 다시 알림]** 버튼을 클릭하여 10분 후에 한 번 더 알람을 받을 수 있습니다.
* **목록 토글 & 테스트 버튼**: 
  - 각 알람은 개별 스위치로 쉽게 활성화/비활성화할 수 있습니다.
  - 종 모양 아이콘(**테스트 알림 받기**)을 누르면 실시간으로 알림 디자인과 이동 링크가 정상 작동하는지 즉시 검증할 수 있습니다.
* **감각적인 다크 테마 UI**: 모던한 폰트(Outfit & Inter)와 은은한 그라데이션, 반응형 트랜지션을 결합한 프리미엄 다크 디자인을 적용했습니다.

## ⚙️ 설치 방법 (Installation)

1. 크롬 브라우저를 열고 주소창에 `chrome://extensions/`를 입력해 이동합니다.
2. 우측 상단의 **개발자 모드 (Developer mode)** 스위치를 활성화합니다.
3. 좌측 상단의 **압축해제된 확장 앱을 로드합니다 (Load unpacked)** 버튼을 클릭합니다.
4. 익스텐션 소스 코드가 위치한 폴더(`D:\workspace\TodoAlarm`)를 선택하여 로드합니다.
5. 크롬 툴바 우측 상단의 퍼즐 조각(확장 앱) 아이콘을 눌러 **할일 알리미 (TodoAlarm)**를 핀으로 고정해 고정 도구에 등록합니다.

## 💡 사용 가이드 (How to Use)

### 1. 알림 추가하기
1. 툴바의 **TodoAlarm** 아이콘을 클릭하여 팝업창을 엽니다.
2. **[+ 알림 추가]** 버튼을 누릅니다.
3. 알림 이름, 본문 메시지, 타겟 사이트 URL, 예약할 요일(칩 클릭), 알림 시각을 입력합니다.
4. **[저장하기]** 버튼을 누르면 스케줄에 반영되며 백그라운드 서비스 워커와 즉시 연동됩니다.

### 2. 알림 테스트하기
* 저장된 카드 목록 중 테스트하고 싶은 항목 하단의 **종 모양 아이콘**을 클릭하면 화면에 즉시 브라우저 알림창이 생성됩니다.
* 알림창 내부의 **[사이트 열기]** 버튼이나 알림 본문을 누르면 설정된 사이트 링크로 안전하게 새 탭이 열립니다.

## ⚠️ 문제 해결 (Troubleshooting)

### Q. 새로고침 후 테스트 알림이 울리지 않아요!
* **원인**: 크롬 개발자 페이지에서 익스텐션을 새로고침(Reload)하면 기존에 켜져 있던 팝업 창의 컨텍스트 연결이 완전히 끊어집니다.
* **해결**: 열려 있는 팝업 창을 닫고 **마우스 클릭으로 팝업을 다시 실행한 후** 테스트 버튼을 클릭해 주세요.

### Q. 설정한 시간에 알림이 오지 않아요!
* **원인 1**: 크롬 스케줄러(`chrome.alarms` API)는 배터리 소모와 브라우저 메모리 부하 방지를 위해 **최소 1분 미만의 알람은 1분으로 강제 보정**합니다. 시각 설정 시 현재 시각보다 최소 2~3분 뒤로 여유롭게 설정하고 테스트해 보세요.
* **원인 2**: Windows 등 운영체제의 **[방해 금지 모드]** 혹은 **[집중 지원]**이 활성화된 상태에서는 알림이 화면에 팝업되지 않고 우측 하단 시스템 알림 센터로 자동 수납됩니다. 방해 금지 모드를 끄거나 알림 센터 내역을 확인해 보세요.
